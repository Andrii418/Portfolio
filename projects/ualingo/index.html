Cel projektu

Nauka języka
przez zabawę

UAlingo to natywna aplikacja Android do nauki języka ukraińskiego, zbudowana w Javie z wykorzystaniem Firebase. Projekt był realizowany na Akademii Tarnowskiej jako kompleksowa aplikacja mobilna.

Inspirowana Duolingo — łączy quizy z timerem, gry słowne, memy z autentycznym językiem ukraińskim i system postępu, który motywuje do codziennej nauki.

Architektura

Jak to działa

01
Logowanie Firebase

Obsługa email/hasło oraz Google OAuth 2.0 przez Firebase Authentication. Po zalogowaniu sesja jest persystowana — użytkownik nie musi logować się przy każdym uruchomieniu. 

mAuth.signInWithEmailAndPassword(
  email, password)
  .addOnCompleteListener(task -> {
    if (task.isSuccessful())
      startActivity(mainIntent);
  });
02
Quiz — timer 60 sekund

CountDownTimer odlicza 60 sekund na każde pytanie. Po upływie czasu odpowiedzi są blokowane. Animacja kolorów ValueAnimator wizualizuje poprawną/błędną odpowiedź. 

new CountDownTimer(60000, 1000) {
  public void onTick(long ms) {
    timerText.setText(
      "" + ms / 1000);
  }
  public void onFinish() {
    disableAnswers();
  }
}.start();
03
Animacja odpowiedzi

ValueAnimator płynnie zmienia kolor tekstu odpowiedzi z białego na zielony (poprawna) lub czerwony (błędna) w czasie 500ms. 

ValueAnimator anim =
  ValueAnimator.ofObject(
    new ArgbEvaluator(),
    fromColor, toColor);
anim.setDuration(500);
anim.start();
04
Przełączanie języków

Oddzielne Activities dla wersji ukraińskiej i angielskiej. Przycisk flagi w menu głównym startuje nową Activity z odpowiednim językiem. 

flagBtn.setOnClickListener(v -> {
  Intent i = new Intent(this,
    EngActivity.class);
  startActivity(i);
  finish();
});
05
Word Search — siatka 7×6

Interaktywna plansza z literami cyrylicy. Użytkownik zaznacza litery po kolei — po znalezieniu słowa licznik się zwiększa i słowo zostaje podświetlone. 

// Znaleziono słów: 0
// GridView 7 kolumn × 6 wierszy
// Litery: cyrylica ukraińska
// Walidacja poziom/pion/ukos
06
18 Activities — nawigacja

Dolny pasek nawigacyjny z 3 ikonami (Dom, Gry, Profil) dostępny na każdym ekranie. Intent-based routing między wszystkimi 18 Activities. 

// Bottom nav → 3 ikony
// Dom    → UkrActivity
// Gry    → WordSearchActivity
// Profil → SettingsActivity

Instalacja

Jak uruchomić

Wymagania
Android Studio Flamingo+ JDK 11+ Gradle 8.0+ Android 6.0+ (API 23)
1
Klonuj repozytorium
git clone https://github.com/IlonaHrabovenko/Android.git
cd Android
2
Skonfiguruj Firebase
# Pobierz google-services.json z konsoli Firebase
# Umieść plik w katalogu: app/
# Dodaj w strings.xml:
<string name="default_web_client_id">YOUR_ID</string>
3
Zbuduj projekt
./gradlew assembleDebug
4
Zainstaluj na urządzeniu
./gradlew installDebug
# lub otwórz projekt w Android Studio i kliknij Run ▶
