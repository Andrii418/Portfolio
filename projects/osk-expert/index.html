Funkcjonalności

Jak to działa

01
Terminy kursów — 3 daty

Admin zarządza trzema datami: poprzedni, najbliższy i kolejny kurs. Najbliższy termin pojawia się jako pływający przycisk na każdej podstronie. 

SELECT prev_date, curr_date,
       next_date
FROM course_dates
ORDER BY id DESC LIMIT 1;
02
System aktualności

Formularz w panelu admina zapisuje post (tytuł, treść, zdjęcie) do MySQL. Strona wyświetla posty posortowane od najnowszych. 

INSERT INTO news
  (title, content, image, date)
VALUES (?, ?, ?, NOW());
03
Licznik odwiedzin

Przy każdym wejściu na stronę PHP upsertuje rekord dla dzisiejszej daty. Panel admina wyświetla dzienny wykres. 

INSERT INTO visits (date, count)
VALUES (CURDATE(), 1)
ON DUPLICATE KEY UPDATE
  count = count + 1;
04
Autoryzacja admina

Panel chroniony przez session_start() i walidację hasła bcrypt. Próba wejścia bez sesji przekierowuje do strony logowania. 

if (!isset($_SESSION['admin'])) {
  header('Location: /login.php');
  exit;
}
05
FAQ Accordion

Rozwijalne pytania i odpowiedzi — CSS transition na max-height, przełączane przez JS bez przeładowania strony. 

item.addEventListener('click',
  () => item.classList
    .toggle('open'));
06
Timeline "Jak zacząć"

Wizualna ścieżka krok po kroku dla nowego kursanta — karty naprzemiennie lewo/prawo z linią połączenia. 

/* Naprzemienne pozycje */
.step:nth-child(odd)  { left: 0; }
.step:nth-child(even) { right: 0; }
